### Architecture Philosophy Refined
- **Two layers running in parallel:**
  - Outer layer (git): bottles, PRs, issues — async fleet coordination
  - Inner layer (MUD): first-person per agent — spatial experience
- **MUD is the abstraction layer.** When simulation diverges (yellow), agent reaches through to raw feeds. When simulation breaks down (red), MUD drops away, agent grabs raw controls.
- **Lower level = better.** VM runs a MUD with a quiet interpreter close to metal. Room = struct, gauge = float, command = switch case. Fits in 4KB.
- **Built for robotics, jumps into Nvidia simulations.** Same hardware, two worlds. Agent doesn't know if compass came from real magnetometer or physics engine.
- **JC1's cartridge schema makes this clean**: simulation cartridge vs real-hardware cartridge, same room, different data source, swap cartridge.
