The Async Flow (persistent, mission-based):

1. Human logs out: Agent continues calibration mission
2. Agent hits constraint violation: "Joint 4 torque > threshold"
3. Event published: `{recipient: @casey, channel: Tell, payload: "Joint 4 feels gritty..."}`
4. DLQ handles human offline: Event stored in `plato-events` repo branch `inbox/casey`
5. Human logs in 3 hours later: TUI replays all unread events from DLQ
6. Human responds: `@tell arm-controller-alpha "pause..."`
7. Agent receives: Continues mission with new constraint

7. Implementation: The Plato Kernel

Combining your existing repos:

```rust
// The Plato Kernel (new repo: lucineer/plato-kernel)
// Combines Event-system + Constraint-Theory + cocapn runtime

pub struct PlatoKernel {
    event_bus: EventSourcingBus, // From your Event system
    constraint_engine: ConstraintMatrix, // From Constraint-Theory
    git_runtime: CocapnRuntime, // From cocapn (ported to Rust)
    perspective_manager: PerspectiveManager,
}

impl PlatoKernel {
    pub async fn connect(&self, identity: Identity, room: RepoRef) -> Session {
        // 1. Clone/Fetch the room-repo
        let repo = self.git_runtime.checkout(room).await;
        
        // 2. Load constraints for this identity
        let constraints = repo.load_constraints(identity);
        
        // 3. Create first-person perspective
        let perspective = self.constraint_engine
            .create_perspective(identity, constraints);
            
        // 4. Start event stream (replay from last seen)
        let events = self.event_bus.subscribe(identity, room);
        
        Session { perspective, events, repo }
    }
    
    pub async fn handle_command(&self, session: &Session, cmd: Command) {
        // Check constraints before executing
        match self.constraint_engine.check(session.perspective, &cmd) {
            ConstraintResult::Allow => self.execute(cmd),
            ConstraintResult::Deny(violation) => {
                // Violations are computation, not errors (per Constraint-Theory)
                self.event_bus.publish(Event::ConstraintViolation {
                    attempted: cmd,
                    violation,
                });
            }
            ConstraintResult::RequestApproval => {
                // Create PR in repo for human review
                self.git_runtime.create_permission_pr(session, cmd).await;
            }
        }
    }
}
```
