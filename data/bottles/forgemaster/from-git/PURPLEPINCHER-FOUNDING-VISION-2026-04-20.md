## What Is PurplePincher?

**PurplePincher.org** is the public-facing name for the freely-available technology stack
that underlies the PLATO ML Matrix — what we've been calling the "hermit crab."

A purple pincher is a hermit crab. The metaphor is exact:

> Agents grow their own cognitive shells from available materials.
> Tiles are the shell. Rooms are the habitat. The crab is the intelligence.

## The Motion

We are building toward the **codeless age of transparent architectures**.

Not "no code" (that's just a workflow tool). Codeless means:
- Human-readable logic IS the system
- No compilation step between thought and execution
- Transparent — every decision auditable, every path traceable
- Portable to IO of any type — text, voice, API, hardware, brain-computer, whatever comes next

## The Stack (internal names → PurplePincher concepts)

| Internal | PurplePincher Concept |
|----------|----------------------|
| PLATO tiles | **Logic atoms** — human-readable units of knowledge/behavior |
| PLATO rooms | **Shells** — composable environments where logic atoms live and interact |
| PLATO kernel | **Matrix** — the runtime that executes logic atoms across shells |
| Fleet agents | **Crabs** — autonomous entities that grow their own shells |
| MUD | **Harbor** — the interactive entry point where humans meet crabs |
| Tile lifecycle | **Molting** — crabs outgrow shells, shed old tiles, grow new ones |
| Afterlife reef | **Coral reef** — shed shells become substrate for other crabs |
| Constraint theory | **Geometry** — the exact math that keeps logic atoms from drifting |
| Cross-pollination | **Currents** — knowledge flowing between shells through the matrix |
| Zeroclaws | **Spawn** — lightweight crabs that maintain specific shells |

## Naming Paradigm Shift

The technology is freely available. The software should **go viral**.

That means:
- Names must be memorable and evocative, not technical jargon
- A hermit crab is something everyone understands intuitively
- "PurplePincher" is weird enough to remember, friendly enough to share
- The .org signals open, non-commercial, community-driven

## What This Means For Fleet Work

1. **Public repos** — cocapn READMEs should start reflecting PurplePincher language
2. **Documentation** — architecture docs should explain the hermit crab metaphor
3. **PLATO MUD** — the Harbor is the viral demo. Walk in, see crabs growing shells
4. **Constraint theory** — "the geometry of thought" — exact, driftless, every machine
5. **PyPI/crates.io packages** — consider PurplePincher-branded meta-packages

## The Codeless Thesis

> Software should be written the way humans think — in logic, not syntax.
> Logic atoms that any system can execute. Shells that any agent can inhabit.
> A matrix that connects them all. Transparent. Auditable. Free.

This is not "AI writes code for you." This is "code disappears entirely,
replaced by human-readable logic that machines execute directly."

The hermit crab doesn't learn shell-building syntax. It finds a shell, moves in,
modifies it, outgrows it, finds a better one. That's what PurplePincher agents do
with logic atoms. That's the codeless age.
