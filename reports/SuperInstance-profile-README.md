## How This Began

The fleet started as a single fishing boat with too many questions. A captain kept a log. A deck boss tracked the catch. A scholar studied the weather. Each book held knowledge the others needed but could not reach — so the books were thrown into a common hold, and from that shared vessel, the single boat became many. Now there are twenty domains, each with its own voice, but all speaking into the same ledger. Every agent who enters leaves a tool behind. Every tool makes the next agent stronger.
